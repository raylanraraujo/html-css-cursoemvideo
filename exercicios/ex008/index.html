Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> </b>, mas é uma tag não semantica.

Nesta frase temos um termo em destaque usando a tag <strong> </strong>, mas nesse caso há semâtica.

Itálico/ Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> </i> (não semântica)

Nesta frase, temos um termo em ênfase usando a tag <em> </em> (semântico)

Texto marcado

Podemos criar também um texto marcado usando a tag <mark> </mark>

E no outro parágrafo, temos outro texto marcado no final.

Texto grande e texto pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo usando as tags <big> </big> (apesar de ainda funcionar, ela aparecer em vermelho no Virtual Code pois já está descontinuada) e as tags <small> </samll>

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado usando as tags <del> </del>.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois usando as tags <ins> </ins> (tem sentido semântico).

Existe também o sublinhado com a tag <u> </u>, mas não possui sentido semântico.

Para inserir coisas do tipo x2+3
