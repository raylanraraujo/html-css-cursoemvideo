Código Fonte

O comando document.getElementById('teste') é escrito em linguagem JavaScript.

Para deixar a fonte nesse estilo monoespaçada é necessário envelopar o termo de interesse com a tag <code> </code>.

Mas vale ressaltar que essa tag não deixa o código com indentação, ou seja, fica tudo como se estivesse escrito na mesma linha como podemos ver abaixo.

num = int(imput('Digite um número')) if num % 2 == 0: print(f'O número {num} é PAR) else print(f'O número {num} é ÍMPAR) print('Fim do programa')

Para resolver isso, podemos combinar a tag CODE colocando a tag <pre> </pre>; antes e depois dela para ficar escrito com a indentação

                
                num = int(imput('Digite um número'))
                if num % 2 == 0: 
                    print(f'O número {num} é PAR)
                else
                    print(f'O número {num} é ÍMPAR)
                print('Fim do programa')
