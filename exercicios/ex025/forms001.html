Meu formulário de fixação de formulário

exitem dois metodos de envio de dados

1) Get (padrão)

Os dados enviados nos formulários aparecem na url. Quando os dados não forem dados sensíveis (como pesquisas do google, nome, sobrenome). Dessa forma, é possível compartilhar a url.

2) Post

Não aparece na url as informações que foram inseridas no formulário. Mas isso não quer dizer que esse método protege o dado enviado. Usado com dados sensíveis, dados com mais de 3mil caracteres. É possível verificar o que foi enviado Inspecionar > Network > Headers

Observações:

1) Normalmente os formulários são feitos utilizando as tags < DIV >, mas por questões de espaçamentos, usaremos < p >.

2) Na "LABEL" temos o "NOME" e o "ID". O "NOME" é importante para HTML e PHP. O "ID" é importante para o Label (para fazer relaçao com o label) e o JavaScript.
